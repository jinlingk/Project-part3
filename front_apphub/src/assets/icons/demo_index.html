Unicode quote

Unicode It is the most original application mode of fonts on the web page：

  • Support dynamic adjustment of icon size color by font
  • Multi color is not supported by default

notice

Unicode The steps as follows：

Step 1: Copy the generated files @font-face

@font-face {
  font-family: 'iconfont';
  src: url('iconfont.woff2?t=1654514557601') format('woff2'),
       url('iconfont.woff?t=1654514557601') format('woff'),
       url('iconfont.ttf?t=1654514557601') format('truetype');
}

Step 2: Define Use iconfont 

.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

Step 3: Select the corresponding icon and obtain the font code to apply to the page

<span class="iconfont">&#x33;</span>

"iconfont" is the font family of your project. Can be viewed by editing items，default is "iconfont"。

font-class quote

Font class is a variation of the way Unicode is used, mainly to solve the problems of unintelligible writing and ambiguous semantics.

Compared with the use of Unicode, it has the following characteristics：

  • Compared with Unicode, it has clear semantics and is more intuitive to write. It is easy to distinguish what the icon is.
  • Because the class is used to define the icon, when you want to replace the icon, you only need to modify the Unicode reference in the class.

Use following steps：

Step 1: Import the fontclass code generated under the project:

<link rel="stylesheet" href="./iconfont.css">

Step 2: Select the corresponding icon and obtain the class name to apply to the page:

<span class="iconfont icon-xxx"></span>

" iconfont" is the font family of your project. It can be viewed by editing items. The default is iconfont.

  • eyes_hide
    #icon-yanjing_yincang
  • eyes_displays
    #icon-yanjing_xianshi
  • password
    #icon-password
  • user
    #icon-user

Symbol quote

It should be said new way that this is the mainstream in the futurepaper This usage actually makes a SVG set, which has the following characteristics compared with the other two：

  • Multi color icons are supported
  • Through some techniques, support fonts through font-size, color 来调整样式。
  • Poor compatibility
  • The performance of browser rendering SVG is average, not as good as png.

using following steps：

Step 1: Import the symbol code generated under the project:

<script src="./iconfont.js"></script>

Step 2: Add general CSS code (just import it once):

<style>
.icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}
</style>

Step 3: Select the corresponding icon and obtain the class name to apply to the page:

<svg class="icon" aria-hidden="true">
  <use xlink:href="#icon-xxx"></use>
